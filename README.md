# 🚀 Career Growth Management Platform

A full-stack web application to manage your career journey with tasks, skills, goals, and job tracking.

## ✨ Features

- ✅ Task Planner with status tracking (Pending, In Progress, Completed)
- 📊 Skill Tracking with date-based progress calculation
- 🎯 Goal Management with start & end date tracking
- 💼 Job Tracker with application stages (Saved, Applied, Interview, Offer)
- 📈 Dashboard with real-time insights

## 🛠 Tech Stack

- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: MongoDB
- Authentication: JWT
